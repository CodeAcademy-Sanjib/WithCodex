# 🧮 Smart Calculator

A modern, responsive Smart Calculator built using HTML, CSS, and JavaScript.  
This project demonstrates clean UI design, expression parsing, and real-time calculation features.

---

## 🚀 Features

- Perform basic arithmetic operations (+, −, ×, ÷)
- Real-time expression evaluation
- Clean and responsive UI
- Dark mode toggle
- Memory functions (MC, MR)
- Keyboard input support

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
